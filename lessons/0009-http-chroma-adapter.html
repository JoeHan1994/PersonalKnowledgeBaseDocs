接上 HTTP:从浏览器把文件送进库

用例已就位,这一课给它装上"门"(路由)和"硬盘"(Chroma 适配器)。

这一课的小胜利:你能讲清一个 POST /ingest 请求怎样逐层下沉—— 路由(interface)→ 用例(application)→ Chroma 适配器(infrastructure)→ 落盘; 并说出"为什么用例只认 VectorStore 端口、从不 import Chroma"。

先认词 · 本课会反复出现

路由 Route Handler
接住一个 HTTP 请求的函数,如 POST /ingest。它只负责收发,把活转交给用例。住在 interface 层。
请求/响应体 DTO · In/Out
过门时的数据外壳:IngestIn(进)/IngestOut(出)。把 HTTP 的 JSON 和领域对象隔开。
适配器 Adapter
端口的具体实现。ChromaVectorStore 把抽象的 VectorStore 接到真实的 Chroma 上。
组合根 Composition root
唯一"把适配器塞进用例"的地方——你项目里的 deps.py · Container。别处都只认端口。
依赖注入 Depends
FastAPI 的 Depends(get_xxx):路由不自己造用例,而是让容器递给它。便于替换与测试。
持久化 persist_directory
Chroma 的 PersistentClient(path=...):把向量写到磁盘,程序重启数据还在(本地、离线)。

① 端口(domain)——用例只认它,不认 Chroma:

# app/domain/ports.py  ——  追加一个出站端口
@runtime_checkable
class VectorStore(Protocol):
    def upsert(
        self,
        ids: list[str],
        vectors: list[list[float]],
        documents: list[str],
        metadatas: list[dict],
    ) -> None: ...

    def query(self, vector: list[float], k: int) -> list[Match]: ...

② 适配器(infrastructure)——把端口接到真实 Chroma:

# app/infrastructure/persistence/chroma_store.py
import chromadb

class ChromaVectorStore:                      # 实现 VectorStore 端口
    def __init__(self, path: str, collection: str = "kb"):
        client = chromadb.PersistentClient(path=path)   # 落盘、离线
        self._col = client.get_or_create_collection(collection)

    def upsert(self, ids, vectors, documents, metadatas) -> None:
        self._col.upsert(                      # 有则覆盖、无则新增(第 8 课的幂等)
            ids=ids, embeddings=vectors,
            documents=documents, metadatas=metadatas,
        )

    def query(self, vector, k):
        r = self._col.query(query_embeddings=[vector], n_results=k)
        return [Match(id=i, text=d, meta=m)
                for i, d, m in zip(r["ids"][0], r["documents"][0], r["metadatas"][0])]

注意:我们自己用 LLMProvider.embeddings()(本地 Ollama)算好向量再传 embeddings=, 不让 Chroma 替我们嵌入——这样模型由你掌控,完全本地。

③ 路由(interface)——只收发,转交用例:

# app/interface/http/routers/ingest.py
router = APIRouter(prefix="/ingest", tags=["ingest"])

@router.post("", response_model=IngestOut)
async def ingest(
    body: IngestIn,
    service: IngestDocument = Depends(get_ingest_service),
) -> IngestOut:
    n = await service.execute(body.path, body.content)
    return IngestOut(path=body.path, chunks_ingested=n)

④ 接线(组合根)——唯一 import Chroma 的地方:

# app/interface/http/deps.py  ——  Container.__init__ 里追加
store = ChromaVectorStore(settings.chroma_path)         # 适配器
self.ingest = IngestDocument(loaders, chunker, self.chat, store, settings.embed_model)

def get_ingest_service() -> IngestDocument:
    return _container.ingest
