更多 Loader:让 PDF 和网页也能进库

回到最初的痛点——"数据源很乱"。这一课给加载器注册表添两类来源,用例一行不改。

这一课的小胜利:你能在第 7 课的 Loader 模式上, 再插两个加载器——PdfLoader(pypdf 抽文本)和 WebLoader(抓网页正文), 挂进按"扩展名/协议"分发的注册表。 IngestDocument 用例一个字都不用改,新格式自动能存能问。

先认词 · 本课会反复出现

注册表 registry
一张"扩展名/协议 → 加载器"的表:.pdf→PdfLoader、http→WebLoader。加来源=往表里加一行。
分发 dispatch
来一个 path,先看它长什么样(后缀/前缀),再派给对应加载器。用例自己不关心怎么解析。
归一化 normalize
不管 PDF、网页还是 md,加载完都吐出同一种 Document(text + metadata)。下游只认它。
语义层缺失 no semantic layer
PDF 只为"打印好看"而生,正文是绝对定位的字符,没有"这是标题/段落"的标记——所以抽文本天生难。
正文提取 main-content
网页里导航/广告/页脚都是噪声。WebLoader 要去模板,只留正文,别把"登录""Cookie 同意"也存进去。
开闭原则 open/closed
对扩展开放、对修改关闭:加新来源只新增加载器,不去改已经跑通的用例和路由。

一张表,管住所有来源

第 7 课你已经有 Loader 接口(吃 path,吐 Document)。 现在的关键是一张注册表 + 一个 pick_loader(path): 它根据后缀或 http 前缀选加载器。来源再乱,入口只有这一个。

.md / .txt → TextLoader
.pdf → PdfLoader
http(s):// → WebLoader
Document 统一出口

② PdfLoader — pypdf 逐页抽文本,拼成一个 Document:

  
# app/infrastructure/loaders/pdf_loader.py
from pypdf import PdfReader          # pip install pypdf

class PdfLoader:
    def can_load(self, path: str) -> bool:
        return path.lower().endswith(".pdf")

    def load(self, path: str) -> Document:
        reader = PdfReader(path)
        text = "\n".join(page.extract_text() or "" for page in reader.pages)
        return Document(text=text, metadata={"path": path, "pages": len(reader.pages)})
  

  

    PDF 没有语义层,正文是绝对定位的字符——抽出来常带怪空格、页眉页脚。
    先能用,后续再加清洗(去页码/合并断行)。扫描件得另上 OCR,pypdf 抽不出图里的字。
  


  
③ WebLoader — 抓网页、去模板只留正文:

  
# app/infrastructure/loaders/web_loader.py
import httpx
from bs4 import BeautifulSoup     # pip install beautifulsoup4

class WebLoader:
    def can_load(self, path: str) -> bool:
        return path.startswith("http://") or path.startswith("https://")

    def load(self, path: str) -> Document:
        html = httpx.get(path, timeout=20, follow_redirects=True).text
        soup = BeautifulSoup(html, "html.parser")
        for tag in soup(["script", "style", "nav", "footer", "header"]):
            tag.decompose()                       # 去模板:扔掉噪声
        text = soup.get_text(separator="\n", strip=True)
        return Document(text=text, metadata={"path": path, "source": "web"})
  


  
④ 注册表 + 分发 — 加来源只动这一处:

  
# app/infrastructure/loaders/registry.py
LOADERS: list[Loader] = [TextLoader(), PdfLoader(), WebLoader()]

def pick_loader(path: str) -> Loader:
    for loader in LOADERS:
        if loader.can_load(path):
            return loader
    raise UnsupportedSourceError(path)   # 没人认领 → 明确报错,别假装能读

# IngestDocument.execute(path) 内部就调 pick_loader(path).load(path)
# —— 用例无需改动,这就是开闭原则
